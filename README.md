# Paradise Nursery

A modern e-commerce plant shopping application built with React and Redux Toolkit. Paradise Nursery allows users to browse through five categories of plants—Air Purifying, Aromatic Fragrant, Insect Repellent, Medicinal, and Low Maintenance—each offering multiple plant options with detailed descriptions and pricing.

The application features a fully functional shopping cart with real-time quantity management, automatic price calculations, and a responsive UI that updates dynamically. Users can add plants to their cart, adjust quantities with increment/decrement buttons, remove items, and view the total cart amount. The cart icon displays a live item count badge, and "Add to Cart" buttons become disabled after an item has been added, providing instant visual feedback to the user.
